Wizualizacja pola magnetycznego i pola elektrycznego z dipolem 
Zespół : Polina Murashko Zuzanna Zyglińska
Repozytorium zostało stworzone na potrzeby projektu wykladowego z fizyki ogólnej , znajduje sie kod napisany w języku Java (Swing).
Nasz kod przedstawia rotujący dipol elektryczny wraz z dynamicznie obliczanymi liniami pola elektrycznego (E) i okręgami pola magnetycznego (B). Program w czasie rzeczywistym wyznacza wektory natężenia pola metodą superpozycji, rysując szczegółowe linie sił ze strzałkami między ładunkiem dodatnim a ujemnym.
